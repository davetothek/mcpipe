## v0.1.0 (2026-06-10)

Initial release.

### Features

- Plugin-based MCP server framework with cache pipeline
- Built-in plugins: git, docker, filesystem
- Transform pipeline: search, limit, offset, head, tail
- Framework tools: view, handles, reload
- LLM self-authoring: write/read/delete plugins and transforms at runtime
- CLI and MCP server entrypoints
- Configuration via TOML + env vars (XDG-compliant)
- Zero runtime dependencies
